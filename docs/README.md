# USTC 编译原理和技术 2025

## 课程信息

??? info "课程简介"

    本课程旨在介绍编译器构造的一般原理、基本设计方法和主要实现技术，强调形式描述方法和自动生成技术，以及对编译原理和技术的宏观理解，主要包括以下理论内容:
    - 形式语言和自动机理论
    - 语法制导的定义和属性文法
    - 类型系统
    - 语言运行时
    - 程序分析和优化
